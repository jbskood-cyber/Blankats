# BlankATS Typography

La tipografia debe tratarse como parte central del producto.

## Fuente recomendada

Usar Geist Sans o Inter como fuente principal. Manrope queda como alternativa.

## Reglas

- Configurar la fuente global desde el inicio.
- Mantener jerarquia clara de titulos, subtitulos, texto y botones.
- Evitar fuentes del sistema sin criterio.
- Usar interlineado amplio y buen espaciado.
- Priorizar legibilidad movil.

## Objetivo visual

BlankATS debe sentirse limpio, premium y confiable.
