# Codex instructions for BlankATS

Goal: improve the premium mobile UX/UI of the existing MVP.

Repository: jbskood-cyber/Blankats

Reference materials:
- Google Drive folder: BlankATS
- Use the Drive mockups and visual references before changing UI.
- Important Drive sections: 02_UX_UI and 07_Recursos.

Do now:
- Keep the current PDF to IA to PDF flow working.
- Redesign the visible experience using the approved BlankATS brand.
- Use the official document plus blue arrow mark and BlankATS wordmark.
- Configure global typography with Geist Sans or Inter.
- Improve text hierarchy, spacing, cards, buttons and mobile layout.
- Add subtle premium animations using motion.
- Split large UI into reusable components.
- Update metadata from generic AI Studio text to BlankATS.

Do not do now:
- Payments.
- Database.
- Email sending.
- Login.
- Admin dashboard.

Validation:
Run npm run lint and npm run build when possible. Report results honestly.
